How does the HTML5 `data-` attribute work, and give an example of how it might be used?

The data- attribute in HTML5 allows you to store custom data private to the page or application. It provides a way to embed custom data attributes on all HTML elements. These attributes can be used to store information that is not displayed to the user but can be utilized by JavaScript, CSS, or other scripts for various purposes.
